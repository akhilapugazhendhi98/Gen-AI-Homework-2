# Gen-AI Homework 2

## Overview
This repository contains the work for Gen-AI Homework 2.

## Workflow
I chose converting meeting notes into structured summaries and action items. This is a business communication workflow where meeting discussions are turned into clear written updates that teams use to track decisions and execute next steps.

## User
The user is a manager, analyst, or team member responsible for documenting meetings and communicating outcomes to stakeholders.

## Input
The system receives raw meeting notes, which may include bullet points, incomplete sentences, or unstructured text captured during a meeting.

## Output
The system produces a structured written update that can be shared with a team, including key discussion points, action items, owners, and deadlines.

## Why this task is valuable
In business settings, meeting outcomes must be communicated clearly and quickly for teams to act on them. Raw notes are often unstructured and inconsistent, making them difficult to use directly. Automating this process helps generate a clear and consistent first draft, reducing manual effort and the risk of missed follow-ups, while still allowing human review before sharing.
